# LAB [week10-day3] switch-while-char 20260715

### INSTRUCCIONES

1- SWITCH: Preguntas bebidas: “Qué desea tomar? Y números:” 1- Vichy 2- Café 3- Té 4- Zumo de naranja 5- Coca-cola Y respuesta: “aquí tienes tu [zumo de naranja] !” Y un default con “debes escribir un número del 1 al 5”. +EXTRA: Que siga preguntando en bucle while hasta que escriba 0 (salir)

2- WHILE: Una asociación benéfica te envía a pedir caridad. Preguntas "una ayudita?", y si te dicen que sí, preguntas cuánto quieren poner. Si te dicen que no, te deprimes y te vas para tu casa. Al final, tienes que decir cuánta gente te ha ayudado y cuánto dinero has recaudado

3- LABS con el TIPO CHAR:

3.1- Ejercicio switch de las bebidas pero con letras en vez de números : A=agua, C=café, etc Ojo, para pasar todo a mayúsculas, se puede usar la clase envolvente (wrapper class) Character: char opcion = Character.toUpperCase(teclado.next().charAt(0)); //junto o separado +EXTRA: en el anterior de una ayudita, usar char para (s/n) en vez de string

3.2- Bucle while "escribe una letra. Para salir pulsa 0". Contar letras hasta que escribes 0 "has escrito 7 letras" (necesita un contador, obviamente)

3.3- Se introduce una frase y se devuelve el texto a la inversa. Ejemplo: olpmejE (ojo, ha de ser letra a letra, no vale usar un método como reverse())
